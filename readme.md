# 📚 Book Scraper

> A Python script that grabs book data so you don't have to.

---

## 🚀 **What It Does**

Visits [books.toscrape.com](http://books.toscrape.com/) and extracts:
- 📖 Book titles
- 💰 Prices
- ⭐ Ratings (1-5 stars)

Saves everything to a clean **CSV file** you can open in Excel.

---

## ⚡ **Quick Start**

```bash
# 1. Install what you need
pip install -r requirements.txt

# 2. Run it
python books.py

# 3. Open books.csv
